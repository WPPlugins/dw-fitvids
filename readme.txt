Easy-to-use jQuery plugin for fluid width video embeds

== Description ==
Do you have publish some YouTube and another video website content in your blog and are looking to make them responsive? This plugin allows you to make the videos responsive and avoid some troubles like recommended videos and notes over videos, a common YouTube problem.

DW FitVids uses FitVids.js originally built by `Chris Coyier` and `Paravel`, it is a lightweight, easy-to-use jQuery plugin for fluid width video embeds.

Demo: Open any post published here: http://www.gamepad.com.br/canais/

Note: Does not forget to check the setup procedure!

== Installation ==
1. Upload the folder `dw-fitvids` to the `/wp-content/plugins/` directory
2. Activate the plugin through the `Plugins` menu in WordPress
3. <strong>Setup</strong>: Go to your `Dashboard` > `Appearance` > `FitVids` and check the jQuery Selector field. For almost every Wordpress installation you can use the selector `.post`
